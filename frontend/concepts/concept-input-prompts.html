Comparison Summary
Variation Graph covered? Feel Complexity
A — Chips No — prompts live below graph Familiar (like ChatGPT suggestions) Low
B — Stacked No — takes up more vertical space Premium, discoverable Low
C — Placeholder No — nothing added above Ultra-minimal, easy to miss Minimal
D — Teal Pills No — compact row Quick, scrollable, tight Low

All variations: Prompts auto-generate from node title + parent. Chips/pills swap out when a different node is focused. When no node is focused, generic prompts show ("What do you do?", "Tell me about yourself"). Placeholder in the input bar also updates to reflect the focused node.
